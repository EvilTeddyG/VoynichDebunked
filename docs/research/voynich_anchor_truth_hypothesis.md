# Research Note: The Anchor-Truth Eclipse Hypothesis
## Leveraging Verifiable Astronomy as a Renaissance Authenticity Signal

## Status

This document is a hypothesis note, not a claim of established fact.

Core claim under evaluation:
- Some astronomical folios in Beinecke MS 408 may have been designed to look consistent with real astronomical computation practice, so that visually verifiable structure could increase perceived authenticity while surrounding text remained low-information filler.

Specific candidate anchor event explored here:
- Total Solar Eclipse of 1433-06-17.

## Abstract

This note proposes an Anchor-Truth hypothesis for the coexistence of:
- low-entropy, recurrence-heavy script behavior, and
- visually structured astronomical folios.

The hypothesis is that a fabricator could embed recognizable, technically plausible astronomical geometry in diagrams (rings, spokes, tracking layouts), while surrounding text remains procedurally generated or semantically weak. In market terms, this would function as a credibility anchor: if the visual astronomy appears mathematically coherent, buyers may infer hidden meaning in the script.

## 1. Why This Hypothesis Is Plausible in Principle

In a high-value manuscript market, a successful fabrication would likely avoid random-looking scientific imagery. A stronger strategy would be:
1. Use diagram geometry that resembles contemporary astronomical instruments/charts.
2. Preserve expected visual density and layout cues (rings, spokes, calibration-like entries).
3. Let visual plausibility carry trust, even if textual substrate is low-information.

This is a generic deception mechanism: mix verifiable structure with non-verifiable claims.

## 2. Candidate Event Anchor: 1433-06-17 Eclipse

Given vellum dating constraints, a major eclipse in-range is a reasonable event class to test as an anchor candidate. The 1433-06-17 eclipse is therefore treated as a concrete test case, not a proven source.

Working proposition:
- If folios f67r/f67v/f68r were tuned to an eclipse-calculation aesthetic, they may preserve geometric features consistent with period planispheric/astrolabe conventions.

## 3. Diagram-Level Features To Test

Potentially relevant visual signals:
- Concentric coordinate tracks.
- Radial spoke partitions consistent with zodiacal framing conventions.
- Central occlusion iconography compatible with eclipse depiction traditions.
- Text-like marks used as calibration stand-ins (density/layout mimicry rather than semantic content).

These are testable at structure level even without asserting direct one-to-one copying from a specific source chart.

## 4. Falsifiable Predictions

The hypothesis gains credibility only if these pass:
1. Geometric fit: radial and angular structure on candidate folios should significantly exceed fit to null-generated pseudo-diagrams.
2. Event specificity: candidate 1433 alignment should outperform nearby-year eclipse candidates under pre-registered scoring.
3. Cross-folio consistency: the same geometric rule-set should explain multiple astronomical folios, not just one cherry-picked panel.
4. Independence from tokenization: textual periodicity claims near diagram regions should not reverse under minor normalization changes.

## 5. Required Evidence Package

Minimum reviewer-grade package:
- Full overlay figures for candidate folios against reconstructed period coordinate grids.
- Null model panel: randomized ring/spoke systems with identical visual complexity.
- Sensitivity analysis over plausible extraction and alignment parameters.
- Family controls: non-Voynich astronomical manuscripts scored under the same pipeline.
- Section holdout results showing stability or failure modes.

## 6. Current Interpretation Rule

If this hypothesis is discussed publicly, the strongest defensible wording is:
- The astronomical folios may function as a visual authenticity anchor, while textual behavior remains more consistent with constrained procedural generation than high-information semantic encoding.

What should be avoided at current evidence level:
- Treating 1433 as proven unique source event.
- Treating any single lag (for example, 13) as privileged without full distribution context.

## 7. Suggested Next Analyses

1. Build a dedicated astronomical-geometry scoring script with pre-registered metrics.
2. Run eclipse-candidate ranking across the vellum date window.
3. Add a figure set that pairs geometric overlays with the already published periodicity/null outputs.
4. Publish all failures and near-misses, not just best-fit candidates.

## Closing

The Anchor-Truth hypothesis is useful because it is falsifiable. Its value is not rhetorical: it either survives systematic geometric and statistical testing, or it does not.
